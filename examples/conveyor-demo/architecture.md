# Conveyor Demo Architecture

The conveyor demo is intentionally small but exercises the core MVP concepts:

- Operator input nodes produce digital signals.
- Safety and sensor nodes can stop equipment and trigger alarms.
- Timer nodes support delayed restart logic.
- Actuator nodes expose runtime state through output signals.
- The debugger records emitted signals as a replayable trace.

The initial workflow is represented as JSON so it can be committed, reviewed, imported, exported, and later generated by visual or AI-assisted tooling.
